# FollowKit

FollowKit is a quote follow-up tracker for small home service businesses.

## Goal

Help small trades and service businesses track quotes they have sent and remember when to follow up.

## Planned features

- User login
- Create and manage quote records
- Track quote status
- Set next follow-up dates
- Save follow-up notes
- Dashboard for due and overdue follow-ups
- Basic reporting

## Tech stack

- Python
- Flask
- SQLAlchemy
- Jinja templates
- Bootstrap
- SQLite

## Local setup

1. Clone the repo
2. Create a virtual environment
3. Install requirements
4. Create a local `.env` file
5. Run the app
